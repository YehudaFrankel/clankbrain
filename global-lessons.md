# Global Lessons — Cross-Project Memory

_Discoveries from any project that apply universally._
_Place this file at: ~/.claude/global-lessons.md (or C:\Users\[you]\.claude\global-lessons.md on Windows)_
_Every project's Start Session reads this file. Discoveries compound across projects._

---

## How to use
1. Copy this file to your home `.claude/` directory (not inside any project)
2. Add `global-lessons.md` to your Start Session skill to read it at session start
3. When /learn finds a lesson that applies beyond the current project, append it here

---

## Format
```
## [YYYY-MM-DD] - [short title]
**Source:** [project name]
**Pattern:** [what works]
**Apply when:** [trigger conditions]
```
